# Fluxborn Module Standard v1.0

## Типы модулей:
1. **Static (Internal):** Собираются внутри, линкуются статически (Math, Core).
2. **Shared (Internal):** Динамические библиотеки (Window, Renderer).
3. **External:** Сторонний код (не трогать руками!).

## Чек-лист создания нового модуля:
- [ ] Создана папка в `Internal/`.
- [ ] Публичные API помечены макросом `UNTIL_API`.
- [ ] Модуль не включает заголовки `src/` других модулей.
- [ ] Все зависимости прописаны в локальном `CMakeLists.txt`.
- [ ] (Опционально) Есть папка `tests/` для этого модуля.

## Запрещенные практики:
- Глобальные переменные (кроме Registry в Core).
- Прямое использование `std::cout` (используй `Until::Logger`).
- Инклюд `Windows.h` или специфичных Linux-заголовков в `.hpp`.
